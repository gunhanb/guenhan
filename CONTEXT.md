# Project Context

This repository contains the source for `burakguenhan.com`, a personal website built with Hugo, Wowchemy, and blogdown-style content.

## Stack

- Hugo Extended `0.101.0`
- Wowchemy Hugo modules
- Netlify deployment via `netlify.toml`

## Key Directories

- `content/`: homepage sections, publications, posts, talks, software, and author metadata
- `config/_default/`: Hugo and Wowchemy configuration
- `static/media/`: downloadable static assets such as PDF files
- `assets/`: theme overrides and static asset sources

## Current Site Structure

- Homepage sections are defined in `content/home/`
- Author/profile content is defined in `content/authors/admin/`
- Publications are stored in `content/publication/`
- Talks are stored in `content/talk/`
- Software entries are stored in `content/software/`
- Blog posts are stored in `content/post/`
